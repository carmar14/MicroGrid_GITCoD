Overview:
En esta carpeta se hace el paso de la microred con controles continuos
a controles usando Z. Esto se hace como paso previo a la adicion de la 
libreira true time.

Para acelerar las pruebas se agrega un script en esta carpeta para hacer
de manara automatizada la discretizacion de los controles resonantes 
y ademas escribir los valores correspondientes en los bloques de simulink.
